## 빌드하기
이 프로젝트는 GUI 시스템이 JavaFX를 이용하므로, JavaFX 런타임이 따로 필요합니다.
매번 설치하는 번거로움을 피하기 위해, JLink를 이용해 플랫폼별 네이티브 앱 형태로 배포합니다.
